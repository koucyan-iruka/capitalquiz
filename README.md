# 首都クイズ
簡易的なPyhonゲームです。  
CSVファイルからランダムで抽出します。  
誤答するまで続きます。  
こちらの<a href="https://note.com/kentoide/n/n16354c4b3458">csvファイル</a>を使用させていただきました
## 遊び方
- レベルを選択します。
  - 1,かんたん  (2択)
  - 2,ふつう　  (4択)
  - 3,むずかしい  (10択)
  - 9,IMPOSSIBLE (首都名を入力)
     - IMPOSSIBLEモードは表記揺れは不正解になります。
     - スキップが5回まで使えます。(空欄で回答するとスキップ）
